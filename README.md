# Automatização das licenças de meter.comm

Este projeto visa automatizar a geração das licenças de Meter.Comm, atividade recorrente do Suporte Técnico. 

Através da biblioteca selenium, o script:

1) acessa o site do Service Now, obtém os dados, 
2) acessa o site do gerador, gera as licenças a partir dos dados
3) depois retorna ao Snow pra responder o chamado e enviar as licenças
